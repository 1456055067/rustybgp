# RustyBGP Documentation

This folder is the consolidated documentation for the RustyBGP codebase.
It is meant to complement the top-level `README.md`, `BUILD.md`, and `DESIGN.md`
files with code-referenced details.

## Index

- `docs/architecture.md` - Threading model and data flow through the daemon.
- `docs/getting-started.md` - Build and run instructions, CLI flags, ports.
- `docs/configuration.md` - Configuration file format and validation rules.
- `docs/grpc-api.md` - gRPC API details and GoBGP compatibility notes.
- `docs/protocols.md` - BGP/BMP/MRT/RPKI protocol support overview.
- `docs/development.md` - Code generation, build internals, and tests.
- `docs/codebase.md` - Module and directory map for quick navigation.

## Related Top-Level Docs

- `README.md` - Project overview and quick start.
- `BUILD.md` - Development environment and CI flow.
- `DESIGN.md` - High-level architectural intent.
